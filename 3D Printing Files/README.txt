These files are 3D printable models used in the assembly of the display unit and the phone attachment.

To produce one display unit, you will need to print:
- 14 x 4-LED Shroud
- 21 x 5-LED Shroud
- 3  x Grid Shroud
- 82 x Tab (if the tab does not fit in the slots within the LED/Grid shrouds, print at 95% scale)
- 1  x Buck Converter Holder
- 1  x HC12 Carriage
- 1  x Buzzer and Switch Mount
- 1  x Arduino Case


To produce one phone attachment, you will need to print:
- 1 x OTG Top
- 1 x OTG Base
- 1 x Each file in the Modular Mounting System
